## 📊 技术栈覆盖

本文档库基于以下技术栈编写，但原理和方法适用于其他技术栈：

### 核心技术
- **前端框架**: Vue 3 + Composition API
- **类型系统**: TypeScript
- **构建工具**: Vite
- **PWA工具**: Workbox
- **状态管理**: Pinia

### 相关技术
- **数据存储**: IndexedDB、LocalStorage、Cache API
- **网络请求**: Axios、Fetch API
- **UI框架**: 现代CSS、响应式设计
- **测试**: Vitest、Cypress
- **部署**: GitHub Actions、Docker
